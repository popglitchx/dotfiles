## What's inside

- i3wm config (with gaps, zenburn colors)
- dunst notifications
- rofi launcher
- kitty terminal
- neovim
- tmux
- copyq clipboard
- flameshot screenshots
- fastfetch
- zsh

## Install

```bash
yadm clone https://github.com/popglitchx/dotfiles.git
```
